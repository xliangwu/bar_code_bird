# bar_code_bird

条形码系统，定制化条形码.

# 功能介绍

用户可以自定制要打印的格式内容，然后批量生产打印内容。

# 语法

> |类型|格式|参数|

- $DATE$yyyy-MM-dd$selectedDate$
- $PARAM$$MACHINE_CODE$
- $TEXT$$MACHINE_CODE$
- $QR_CODE$$$

# 技术栈

- SpringBoot 2.6.2
- Mybatis
- vue3+
